# TARS HVAC Duct Sizer
## Nightly Creative Build - 2026-02-02

A practical Streamlit web app for MEP/HVAC engineers to quickly size ducts.

### Features
- Calculate round and rectangular duct sizes based on CFM and velocity
- Friction loss calculations
- Standard size suggestions
- Interactive sliders and inputs

### Quick Start
```
pip install -r requirements.txt
streamlit run app.py
```

Open localhost:8501 in browser.
